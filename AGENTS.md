## TypeScript Agent Delegation

**TypeScript Expert Agent (REQUIRED)**: All TypeScript implementation, modification, review, or debugging of `.ts`/`.tsx` files MUST be delegated to the `typescript:typescript-expert` agent. This includes:

- Creating new TypeScript files
- Modifying existing TypeScript code
- Type definitions, interfaces, and generics
- React components and hooks
- Zod schemas and validation
- Node.js/server-side code
- React Router loaders/actions
- Drizzle ORM schemas and queries

Use the Task tool with `subagent_type="typescript:typescript-expert"` for all TypeScript work.
